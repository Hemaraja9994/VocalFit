# VocalFit - Vocal Health App

## How to Run (3 steps)

### Step 1: Install Node.js (one-time only)
- Go to https://nodejs.org
- Click the big green "LTS" button
- Run the installer (click Next through everything)
- Restart your computer

### Step 2: Install Expo Go on your phone
- Android: Open Play Store → search "Expo Go" → Install
- iPhone: Open App Store → search "Expo Go" → Install

### Step 3: Start the app
- On Windows: Double-click **START_APP.bat**
- On Mac: Open Terminal, navigate here, run: `npx expo start`
- Scan the QR code with Expo Go on your phone

That's it! The app will load on your phone.

## Troubleshooting
- If QR code doesn't connect: Make sure phone and computer are on SAME Wi-Fi
- If you see "incompatible version": Update Expo Go from the Play Store / App Store
- If npm errors: Run `npm install --legacy-peer-deps` in this folder
